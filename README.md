## Props
### defaultCountry
'RU', 'EN', 'USA', 'FR', etc.
### ~~focusOnSelect~~
Skip flag icon from tab-button navigation
### alwaysShowPrefix
Show prefix only if component has focus 
### compactView
Hide flag if component not in focus
### type
input type. By default `type="tel"`
